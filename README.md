# House-Price-Prediction-using-Machine-Learning

## House Price Prediction using Machine Learning
## Project Overview
This project predicts the selling price of houses based on property features such as area, number of bedrooms, and year built.
The goal is to build a machine learning model that can estimate house prices accurately using historical data.
The project demonstrates a complete machine learning pipeline, including data preprocessing, feature selection, model training, and prediction.

## Problem Statement

Predict the SalePrice of a house using property attributes.

## Dataset

The dataset contains information about houses and their features.
Example features used in this project:
Area – Size of the house
Bedrooms – Number of bedrooms
YearBuilt – Year the house was constructed
SalePrice – Target variable (house price)

## Technologies Used

Python
NumPy
Pandas
Scikit-learn
Google Colab

## Machine Learning Model

The model used in this project:
Linear Regression
Linear Regression learns the relationship between house features and their selling price and predicts prices for new houses.

##  Project Workflow

Import required libraries
Load dataset
Select important features
Split data into training and testing sets
Train the Linear Regression model
Predict house prices on the test data
Convert predicted log values back to real prices
Display predicted results
